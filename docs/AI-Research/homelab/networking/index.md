# Networking for Homelab

A comprehensive guide to understanding home networking from basics to advanced topics. Designed for homelab enthusiasts of all levels.

## Overview

This guide covers networking concepts essential for running a homelab:
- Setting up servers, VMs, and containers
- Configuring network access (VPN, port forwarding)
- Securing your network with VLANs and firewalls
- Troubleshooting connectivity issues

## Prerequisites

- Basic computer knowledge
- Access to a home router (ISP-provided or third-party)
- Curiosity to learn how the internet works!
